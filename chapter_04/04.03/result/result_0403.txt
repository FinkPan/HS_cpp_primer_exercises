 下列哪些定义是错误的？ 
          (a) int ia[7] =  { 0, 1, 1, 2, 3, 5, 8 }; 
          (b) vector<int> ivec = { 0, 1, 1, 2, 3, 5, 8 }; 
          (c) int ia2[ ] = ia1; 
          (d) int ia3[ ] = ivec; 
 
 答:
 (a)	正确	
 (b)	错误,vector没有这样的初始化方式
 (c)	错误,数组不能直接赋值
 (d)	错误,数组不能直接赋值
 
 程序验证:
 (b) “ivec”: 不能用初始值设定项列表初始化非聚合
 (c) “初始化”: 无法从“int [7]”转换为“int []”
 (d) “初始化”: 无法从“std::vector<_Ty>”转换为“int []”
